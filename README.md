# 🏰 Tales From The Tavern

A Dungeons & Dragons web app for creating characters, tracking stats, and managing campaigns — all wrapped in a retro fantasy UI.

## ✨ Features

- 🧙 Create custom characters (race/class/subclass)
- 🗡️ Battle simulator (d20-based)
- 📜 Full login system using ASP.NET Identity
- 🖼️ Upload avatars and backgrounds
- 🎵 Fantasy sound effects and animations

## 🛠 Tech Stack

- ASP.NET Core 8.0 MVC
- Razor Pages
- Bootstrap 5
- Azure SQL (via EF Core)
- Hosted on GitHub + Azure
